INSTALL
-------

  **Depends on photos module and libraries module v2.x.
  *Download swfupload: http://code.google.com/p/swfupload/.
  *Rename folder to swfupload and add to sites/all/libraries.
  *Enable module.


FEATURES
--------

  *Adds support for SWFUpload form to replace HTML image upload form.
  *Upload several images to an album easily with flash upload form.
  *http://code.google.com/p/swfupload/
